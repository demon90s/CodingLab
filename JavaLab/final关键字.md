java中的final关键字的作用：
1、用来修饰一个引用；
2、用来修饰一个方法；
3、用来修饰类。

当final修饰方法时，这个方法会成为最终方法，无法被子类重写。

如果引用为基本数据类型，则该引用为常量，该值无法修改；

如果引用为引用数据类型，比如对象、数组，则该对象、数组本身可以修改，但指向该对象或数组的地址的引用不能修改。

如果引用时类的成员变量，则必须当场赋值，否则编译会报错。

```java
public class Month {
    // 静态常量
    public static final int JAN = 1;
}
```
